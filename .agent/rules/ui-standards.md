# Rule: NRI-Luxury Aesthetic

## Color Palette
- **Background**: `#FFFFFF` (Clean, crisp)
- **Secondary**: `#F9FAFB` (Soft gray for depth)
- **Accents**: 
    - `Emerald-700` (#047857): Representing growth, safety, and lifestyle match.
    - `Indigo-900` (#312E81): Deep slate/trust color for structured data.

## Typography
- **Font Family**: Sans-serif only. Priority: `Geist`, `Inter`, then system-ui.
- **Headlines**: `font-semibold` with generous tracking (e.g., `tracking-tight` or `tracking-wide` depending on context).
- **Body**: High readability, `text-slate-600`.

## The "Airbnb" Rule (Card Design)
- **Images**: Must take up **60%** of any property card.
- **Border Radius**: strict `12px` (rounded-xl) on all cards.
- **Borders**: Minimalist. `border-none` preferred with `shadow-lg`, or very subtle `border-slate-100`.

## Visual Hierarchy
1. **City Score / Match %**: Highest priority.
2. **School Commute**: Second priority (the "Why").
3. **Square Footage/Price**: Tertiary (initially hidden or smaller).

## Iconography
- Use `lucide-react` vectors.
- Stroke width: `1.5px` (Thinner, more elegant).
